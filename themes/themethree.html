Theme 04: Basic Animation

Assignment: 04.01.01 Sprite animation on a background


Goal: For this assignment, we had to develop a character based on our research and moodboard. Afterwards, to create a corresponding character’s visual design and background, and then employ animation principles to create walk cycles. Here's the character Mr. Pupsky in action.

My part: All elements such as sketching, illustrations and coding were done by me.

Key takeaways: The assignment helped learn how to reflect the character’s identity visually, create an illustration based on sketching and how to use CSS so that the background is adjusted to the screen and character moving throughout it.

Assignment: 04.02.01 Javascript controlled animation


Goal: To use SVG animation and with the help of CSS create 10 different animation functions that they can be switched on and off with javascript.


My part: As it was an individual work, everything was done by myself.


Key takeaways: I’ve acquired additional knowledge about CSS animation, and its variations such as spinning, skewing and the like. Besides this, I learnt the basics of javascript, especially when it comes to the button control so it enacts the desired animation.

Assignment: 04.03.01 Project pool 1

Goal: To create a page where based on the user’s submitted answers the fortune teller randomly selects one of the predefined answers.

My part: Within a group, my focus area was to create content and assisting with coding.

Assignment: 04.04.01 Link to your Onboarding

Goal: To create a web-based onboarding for potential foreign students of KEA, to help them with a chosen area of importance. In our case it was, explaining the process of paperwork and what is necessary to reside and get employed in Denmark, enhancing this message with various multimedia and gamification elements.

My part: The assignment was group work, so my role was jointly developing the idea, research of the information needed for the game; moodboard and persona collage creation, written content, and design elements.

Key takeaways: Learning how to organize a larger-scope project based on everyone’s strengths, polishing my writing and Illustrator skills, in particular when designing elements for the game.
